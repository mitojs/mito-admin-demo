## 概览
mito平台错误详情数据展示，利用`msw（mock service work）`在浏览器本地搭建服务mock接口

### 安装
`npm i`
### 启动
`npm run dev`

### Vue示例
http://localhost:1024/#/errors/1/info


### React示例
